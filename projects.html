Modern Network Security Applications (2023-2024)

Used Linux Virtual Machine Manager and the services of Cyclades Okeanos Knossos(provided me with the Virtual Machine that I used as a server to perform the tasks) to set up and configure a virtual machine running Debian 12. In addition to the setup, I performed various security tasks:

  • Computer Security: Implemented a robust firewall using iptables to safeguard the network from unauthorized access and malicious traffic.
  • Remote Access Security: Installed and configured fail2ban to proactively prevent unauthorized SSH access attempts, enhancing overall remote access security.
  • Denial-of-Service (DoS) Mitigation: Conducted thorough testing for DoS attacks using tools such as tcpdump, port scan, netstat, and netcat to identify vulnerabilities and fortify the network against potential attacks.
  • File Encryption/Decryption: Implemented file encryption techniques to ensure secure transmission of sensitive data, enhancing data confidentiality. Additionally, I verified the authenticity of received files through decryption processes.
  • DNS Attacks Simulation: Created controlled environments of virtual machines to simulate malicious DNS attacks, including hijacking and pharming attacks, to assess the network's resilience against such threats.
  • Web Application Security: Developed a basic website and conducted comprehensive security assessments using tools such as Mozilla Observatory, CryptCheck, Security Headers, and Qualys SSL Labs. Achieving a high score of 115/100 on Mozilla Observatory demonstrates the robust security measures implemented.

Computer Networks (2022)

For my project on Computer Networks, I utilized GNS3 to emulate diverse network environments, consisting of Cisco routers, Cisco switches, and Windows/Linux clients. Alongside network emulation, I performed various additional tasks:

  • Network Diagnostics: Leveraged traceroute and ping utilities to diagnose and troubleshoot network connectivity issues, ensuring seamless communication within the network infrastructure.
  • Network Stability: Configured virtual machines with static IP addresses to maintain network stability and prevent IP conflicts, enhancing overall network reliability.
  • DHCP Binding: Implemented DHCP binding to automate IP assignment, simplifying network administration tasks and optimizing IP management.
  • Maintenance: Conducted routine maintenance tasks to ensure the network's reliability and performance, including software updates, configuration audits, and performance optimizations.
